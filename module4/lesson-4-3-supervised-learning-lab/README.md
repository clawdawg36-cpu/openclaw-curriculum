# Lesson 4.3 — Supervised Learning Lab

**Module:** 4 — Give Your Agent the Ability to Learn  
**Duration:** 2 periods (100–120 minutes total)  
**Grade Level:** 6–8

---

## Overview

Students move from understanding ML to doing ML. Using Google's Teachable Machine, they design a classification task, collect training data, train a model, and evaluate its performance across easy, edge, and adversarial test cases. The two-period structure gives time for both training and genuine iterative improvement.

## Key Activity

**Teachable Machine Lab** — Students train a real image or audio classifier using webcam/microphone input. Structured evaluation (easy / edge / adversarial test sets) builds systematic thinking about model performance. The iteration step (add data, retrain, compare) makes the data → accuracy relationship visceral.

## Requirements

- Computers with webcam access (one per student or pair)
- Browser access to teachablemachine.withgoogle.com
- **Test tech access before class day**

## Essential Questions

- What is supervised learning and how does it differ from other ML approaches?
- Why do training accuracy and test accuracy often differ?
- What does it mean to iterate on a model?
