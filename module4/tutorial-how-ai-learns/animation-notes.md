# Animation Notes — "How AI Learns"
**For motion graphics / animation team**

---

## Style Guide
- Color palette: Bright blues, greens, corals, purples — high contrast, accessible
- Characters: Simple 2D flat design, robot mascot with expressive face
- Typography: Rounded sans-serif (e.g., Nunito or Poppins), large and readable
- Transitions: Smooth slides and fades, ~0.4s easing

---

## Key Animation Sequences

### 1. Opening (0:00–0:40)
- Robot character surrounded by floating images (cats, dogs, pizza, bikes)
- Images shuffle randomly — robot looks confused, question marks pop up
- Color-coded bubbles appear around each image type

### 2. Traditional vs ML Diagram (0:40–1:40)
- Split screen: left side (blue) = traditional programming flowchart
- Right side (green) = ML flowchart with arrow pointing from Data+Labels → Model
- Animated arrows showing the "flip"

### 3. Three Doors (1:40–3:00)
- Three colorful doors with icons (chalkboard, puzzle, game controller)
- Each door opens to reveal the scene described in script
- Sound cue: door creak on open

### 4. Training Conveyor Belt (3:00–5:30)
- Conveyor belt carrying labeled image tiles into a glowing "Model Machine"
- Machine whirs, processes, outputs a percentage accuracy gauge
- Accuracy bar animates: 30% → 60% → 85% → 94% with bouncy easing

### 5. Neural Network (5:30–6:30)
- Animated layers of nodes with pulsing connections
- Color highlights flow left-to-right showing data moving through layers
- Labels: "Input Layer," "Hidden Layers," "Output Layer"

### 6. Dart Board (6:30–7:15)
- Overfitting: Darts cluster in one spot (not the bullseye)
- Underfitting: Darts scattered all over
- Just right: Darts near bullseye, reasonable spread

### 7. Ending Badge (7:15–8:00)
- Robot from opening now confidently sorts animals into correct bins
- "ML Engineer" badge pulses with gold animation
- Three action item cards slide in from right

---

## Accessibility Notes
- All text on screen for minimum 3 seconds
- High contrast text (≥4.5:1 ratio)
- No flashing content faster than 3Hz
- Captions file: captions.vtt
