# Module 4: Student Activity Materials

All student-facing activity materials for Module 4 — AI & Machine Learning Foundations.

## Materials Overview

| File | Activity | Format | Time |
|------|----------|--------|------|
| `01-human-learning-machine-game-guide.md` | Human Learning Machine — whole-class simulation | Print / Digital | 45–55 min |
| `01b-facilitator-cards.md` | Facilitator & role cards for the simulation | Print (cardstock) | — |
| `02-labeling-sprint.md` | Labeling Sprint — 20 face images + recording sheet | Print / Digital | 20–30 min |
| `03-overfitting-demo-worksheet.md` | Overfitting Demo — Teachable Machine 3 vs 30 images | Digital + Print | 30–40 min |
| `04-kmeans-clustering-activity.md` | K-means Clustering — mystery data + hand-clustering | Print / Digital | 30–40 min |
| `05-lab-report-template.md` | Lab Report — sentiment classifier performance | Print / Digital | 45–60 min |
| `06-ethics-essay-prompt.md` | Ethics Essay — AI trained on one city's data | Print / Digital | 45–60 min |

## Sequence

Activities are designed to be used in order, but each can stand alone:

1. **Human Learning Machine** — experiential intro to training feedback loops
2. **Labeling Sprint** — hands-on data annotation, inter-rater reliability
3. **Overfitting Demo** — empirical investigation with Teachable Machine
4. **K-means Clustering** — unsupervised learning through spatial reasoning
5. **Lab Report** — synthesis and evaluation of classifier performance
6. **Ethics Essay** — critical reflection on bias and representation

## Print Notes

- **01-human-learning-machine-game-guide.md** — print one copy for facilitator
- **01b-facilitator-cards.md** — print on cardstock and cut; laminate for reuse
- **02-labeling-sprint.md** — one copy per student
- **03-overfitting-demo-worksheet.md** — one copy per student; requires laptop/Chromebook
- **04-kmeans-clustering-activity.md** — one copy per student; graph paper optional
- **05-lab-report-template.md** — one copy per student or pair
- **06-ethics-essay-prompt.md** — one copy per student

## Accessibility

All materials are designed to be:
- Screen-reader friendly in markdown format
- Convertible to PDF for printing
- Usable without proprietary software (except where Teachable Machine is specified)

## Standards Alignment

These materials support:
- CSTA K-12 CS Standards (Data & Analysis, AI literacy)
- Common Core ELA (argumentative writing, evidence-based analysis)
- NGSS Science & Engineering Practices (data collection, analysis, communication)
