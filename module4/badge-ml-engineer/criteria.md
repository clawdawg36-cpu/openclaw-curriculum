## Badge Description

> Awarded to students who demonstrate understanding of the machine learning pipeline — including data collection, labeling, model training, and evaluation — and can apply these concepts using a real ML tool.

---

## Earning Criteria

Students must complete **ALL** of the following to earn the ML Engineer badge:

### ✅ Required Completions

| # | Requirement | Evidence |
|---|-------------|----------|
| 1 | Watch "How AI Learns" animated video (Module 4) | Completion recorded in LMS |
| 2 | Complete the drag-and-drop training simulator (all 3 phases: label → train → test) | Simulator completion flag |
| 3 | Complete the Machine Learning for Kids walkthrough (train and test a model) | Screenshot or teacher verification |
| 4 | Pass the Module 4 ML Concepts quiz with **≥75% (6/8 correct)** | Quiz score ≥ 75% |
| 5 | Post in the async forum discussion with a substantive original response | Discussion post submitted |

---

## Mastery Indicators

A student who earns this badge can:

- **Explain** the difference between supervised, unsupervised, and reinforcement learning
- **Describe** the four-stage supervised learning pipeline (collect → label → train → test)
- **Define** key terms: training data, labels, epoch, accuracy, overfitting, underfitting, neural network
- **Use** a beginner ML platform (Machine Learning for Kids) to train and test an image classifier
- **Analyze** the effect of data quality and quantity on model performance
- **Reflect** on real-world implications and ethical considerations of machine learning

---

## Optional Extensions (for "ML Engineer ⭐ Level Up"!)

Students who complete the following may earn an extended badge annotation:

- [ ] Add ≥30 training images per category in ML for Kids and achieve >90% test accuracy
- [ ] Build a Scratch project using their trained ML model
- [ ] Write a 1-page reflection: "One way AI could help my community — and one risk I'd want to prevent"
- [ ] Present their trained model to the class and explain what it learned
