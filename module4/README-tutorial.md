## Learning Objectives — Module 4

By the end of this module, students will:
1. Explain what machine learning is and how it differs from traditional programming
2. Identify the three types of ML: supervised, unsupervised, reinforcement
3. Walk through the four-stage supervised learning pipeline
4. Apply ML concepts by training a real model in Machine Learning for Kids
5. Reflect on real-world applications and ethical implications
