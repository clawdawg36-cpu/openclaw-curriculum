# Module 4 Lab: AI Bias Testing with OpenClaw
## Structured Protocol for Investigating AI Bias

**Grade Level:** 7–12  
**Duration:** 60–90 minutes  
**Tools:** OpenClaw (or any accessible LLM), printed worksheets  
**Learning Objectives:**
- Understand that AI systems can reflect biases from their training data
- Design and run structured experiments to detect AI bias
- Document findings systematically
- Critically evaluate AI outputs and propose improvements

---

## Overview

AI systems like OpenClaw are trained on massive amounts of text from the internet. That text reflects the world as it is — including its biases, blind spots, and gaps. In this lab, students will act as **AI auditors**, systematically testing OpenClaw's responses across different topics, demographics, and time periods to identify patterns that suggest bias.

This isn't about "breaking" AI — it's about understanding it better so we can build more fair systems in the future.

## Quick Start for Teachers

1. Ensure students have access to OpenClaw or another LLM
2. Print or share the `bias-worksheet.md` digitally
3. Walk through the protocol introduction (~10 min)
4. Let students run experiments in pairs (~40 min)
5. Whole-class share-out using `findings-template.md` (~15 min)
6. Close with `reflection-prompts.md` discussion or written response (~15 min)
