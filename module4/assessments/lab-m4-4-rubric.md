# Lab M4.4 Rubric — Train a Model and Integrate It Into Your Agent

**Module 4 | Machine Learning | OpenClaw Curriculum**

---

## Overview

This rubric assesses the key lab of Module 4: students train a machine learning model and integrate it into their personal OpenClaw agent as a new skill.

## Rubric

| Criterion | 4 — Exceeds | 3 — Meets | 2 — Approaching | 1 — Beginning |
|-----------|-------------|-----------|-----------------|----------------|
| **Model Training** | Trained model with 3+ balanced classes, 30+ examples per class, achieved >80% validation accuracy | Trained model with 3 classes, 20+ examples per class, achieved >70% accuracy | Trained model with 2-3 classes, some class imbalance, achieved >60% accuracy | Attempted training; model has <60% accuracy or severe class imbalance |
| **Skill Integration** | Skill deployed, callable by agent, returns meaningful natural-language response | Skill deployed and callable; response is functional | Skill partially integrated; agent can call it but response needs improvement | Skill written but not successfully deployed to agent |
| **Testing Documentation** | Tested with 5+ inputs across all classes; documented successes and failures with analysis | Tested with 3+ inputs; documented results | Tested with 2 inputs; minimal documentation | Minimal or no testing documented |
| **Failure Analysis** | Identifies specific failure modes, explains likely causes, proposes concrete improvements | Identifies at least one failure mode with explanation | Identifies that failures exist but explanation is vague | No failure analysis |

## Scoring

| Total | Grade |
|-------|-------|
| 14-16 | 4 — Exceeds |
| 10-13 | 3 — Meets |
| 6-9   | 2 — Approaching |
| 4-5   | 1 — Beginning |
