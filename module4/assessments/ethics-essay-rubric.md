# Ethics Essay Rubric
## Module 4 — Short Ethics Essay: "Only Trained on One City"

> **Essay Prompt:**
>
> *You have trained a sentiment classifier using data collected from one city — let's call it Riverdale. Your agent is now being used to classify the mood of social media posts from across the entire country.*
>
> *Write a short essay (2–3 paragraphs, 200–350 words) arguing: **Is it fair to use this classifier nationwide? What problems might arise? What would need to change to make it more fair?***
>
> *Your essay must take a clear position, support it with reasoning, and use at least one real-world example (from class or your own research) to make your argument concrete.*

## 📝 Sentence Starters (for students who need a jump-start)

- *"Using a classifier trained only on Riverdale data nationwide is [fair/unfair] because…"*
- *"When a model is trained on data from only one place, it learns [patterns / language / expressions] that may not apply to…"*
- *"A real-world example of this kind of bias is [example], where…"*
- *"To make this classifier fairer, developers would need to…"*
- *"The core ML concept at play here is [distribution shift / class imbalance / training data bias], which means…"*

---

## 🧠 Key Vocabulary for This Essay

| Term | What it means for this essay |
|---|---|
| **Training data** | The examples the model learned from — only from Riverdale |
| **Distribution shift** | When real-world data looks different from training data |
| **Generalization** | Whether a model that learned from one place works somewhere else |
| **Bias (in ML)** | When a model performs worse for some groups than others |
| **Class imbalance** | When one type of example is overrepresented in training |
| **Overfitting** | When a model is too tuned to its training examples to work elsewhere |
