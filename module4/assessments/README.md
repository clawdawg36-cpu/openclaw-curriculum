# Module 4 Assessments
## Give Your Agent the Ability to Learn

This folder contains all assessment tools for Module 4.

| File | What it is |
|---|---|
| `sentiment-classifier-lab-report-rubric.md` | 5-dimension rubric for the Sentiment Classifier Lab Report (performance analysis + improvement ideas). 20-point scale. |
| `ethics-essay-rubric.md` | 4-dimension rubric for the short ethics essay: "Only Trained on One City" prompt. Assesses claims, reasoning, examples, and writing. 16-point scale. |
| `exit-tickets.md` | Daily exit ticket templates for Lessons 4.1–4.5. 3–5 min formative checks with stretch questions. |
| `quiz-ml-concepts-answer-key.md` | Teacher answer key for the ML concepts quiz. 5 questions, Recall → Analysis difficulty. Includes common misconceptions. |
| `portfolio-prompt.md` | Student-facing portfolio prompt. Covers both the lab report and ethics essay as a combined Module 4 artifact. |
