# Module 4 Lab: Sentiment Classifier
## Machine Learning for Kids + Scratch

**Grade Level:** 6–10  
**Duration:** 90–120 minutes (can split across two class periods)  
**Tools:** Machine Learning for Kids (machinelearningforkids.co.uk), Scratch 3  
**Learning Objectives:**
- Understand supervised machine learning through hands-on training
- Train a text classifier to detect positive/negative sentiment
- Build an interactive Scratch project using a trained ML model
- Evaluate model performance on unseen data

---

## Overview

In this lab, students will train a machine learning model to classify movie reviews as **positive** or **negative** using the free Machine Learning for Kids platform. They'll then connect that model to a Scratch project that can automatically sort new reviews — without anyone programming the rules manually.

This is real machine learning: you teach the AI by showing it examples, and it figures out the patterns on its own.

## Quick Start for Teachers

1. Create a class account at [machinelearningforkids.co.uk](https://machinelearningforkids.co.uk)
2. Set up student logins (or use the "Try it without logging in" option)
3. Distribute `student-lab-guide.md`
4. Allow ~30 min for training, ~45 min for Scratch building, ~15 min for testing
5. Assign `lab-report.md` as homework or in-class wrap-up
