# Feature Extraction Game — Object Description Cards

**Lesson:** 3.1 — From Data to Knowledge  
**Purpose:** Students practice turning real-world objects into numerical feature vectors, then compare results with classmates.

---

## How to Play

1. Each student (or pair) receives one **Object Card** below.
2. Using the **Feature Template**, assign a number to each feature for your object.
3. When time is up, read your numbers aloud — don't say the object name!
4. The class tries to **guess** which object you encoded.
5. Compare: did other students encode the same object differently?

---

## Feature Template

Use this template for every object. Rate each feature on the scale shown.

| # | Feature | Scale | Your Score |
|---|---|---|---|
| 1 | **Size** | 1 (tiny) → 10 (huge) | |
| 2 | **Weight** | 1 (feather-light) → 10 (very heavy) | |
| 3 | **Furriness / Texture softness** | 0 (hard/smooth) → 10 (very fluffy/soft) | |
| 4 | **Color brightness** | 0 (black/dark) → 10 (white/neon bright) | |
| 5 | **Makes a sound on its own** | 0 (silent) → 10 (very loud) | |
| 6 | **Has legs** | 0 = no, 1 = yes | |
| 7 | **Is alive (or was once alive)** | 0 = never alive, 1 = alive or was alive | |
| 8 | **Is edible** | 0 = not edible, 1 = edible | |
| 9 | **Has a sharp point or edge** | 0 = no, 1 = yes | |
| 10 | **Roundness** | 0 (flat/angular) → 10 (perfectly round) | |

**Discussion questions:**
1. Which two objects had the most similar feature vectors? Would a computer confuse them?
2. Which features were hardest to score? Why?
3. If you could add one more feature to the template, what would it be? Would it help separate any "hard to tell apart" objects?
4. What would happen if you gave the computer *only* the "has legs" and "is edible" features? How much would it get right?

---

## Extension Challenge

**Can you fool the machine?**

Pick two very different objects (e.g., a cat and a marshmallow). Try to come up with a feature vector that would make a computer think they're the same thing. Which features did you have to manipulate? What does this tell you about the limits of feature vectors?
