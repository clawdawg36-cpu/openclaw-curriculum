# 🃏 Pattern Detective — Data Cards

**Module 3 | Print and Cut (or Display Digitally)**

---

## How to Use These Cards

**Print and cut:** Each card below is one puzzle. Cut them apart and distribute 2–4 per student or group.

**Digital display:** Project one card at a time on the board for whole-class warm-ups.

**Difficulty levels:**
- 🟢 **Green** — Straightforward. Good starting points.
- 🟡 **Yellow** — Requires a bit more thought. Good for pairs.
- 🔴 **Red** — Tricky or has a twist. Good for challenge groups.
- 🚨 **Trick Cards** — Contains a deliberate error. Use sparingly; great for discussion.
