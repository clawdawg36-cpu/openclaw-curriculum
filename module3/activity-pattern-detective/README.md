# 🔍 Activity: Pattern Detective

**Module 3 | Warm-Up Card Activity**
**Time:** 10–15 minutes (warm-up) or 20–30 minutes (full activity)
**Format:** Individual, pairs, or small groups

---

## What Is This Activity?

Students become Pattern Detectives — analyzing sets of data cards, identifying the hidden rule that connects them, and explaining *why* finding that rule matters for AI.

This activity is designed as a **warm-up** for Lesson 3.3 (Pattern Recognition), but works equally well as a standalone exploration or exit ticket.

The core insight students should leave with:

> **"AI doesn't memorize — it finds rules. If it finds the wrong rule, it makes wrong predictions. That's why data quality and pattern clarity matter."**

## Learning Goals

By completing this activity, students will:
- Identify hidden rules in number sequences, data tables, and visual patterns
- Explain why finding the *rule* matters more than just knowing the next value
- Analyze what happens when an AI learns the wrong pattern
- Connect pattern recognition to how AI agents generate responses and make predictions

## Quick Start

**Warm-up mode (10–15 min):** Distribute 2–3 cards per student or group. Students identify the rule and write a one-sentence explanation. Discuss as a class.

**Full activity mode (20–30 min):** Students work through 4–5 cards individually, then complete the reflection in `activity-guide.md`. Includes a "broken pattern" analysis and connection to AI.

Teachers: see `teacher-guide.md` for facilitation tips and answer explanations.
