## Key Terms Covered

- **Pattern** — A regularity or rule that repeats across many examples in data
- **Inference** — Drawing a conclusion about new data based on learned patterns
- **Chain-of-thought** — Prompting technique that asks AI to reason step by step
- **Hallucination** — AI generating confident but false information
- **Explainability** — The degree to which an AI system can explain its reasoning
- **Black box** — An AI system that produces outputs without being able to explain how
