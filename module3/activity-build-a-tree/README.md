# 🌳 Activity: Build-a-Tree

**Module 3 | Hands-On Activity**
**Time:** 25–35 minutes
**Format:** Individual, pairs, or small groups

---

## What Is This Activity?

Students build decision trees by hand — starting from a real-world classification problem (animals, weather, food) and working out the yes/no questions needed to sort things into categories.

This is the foundational hands-on complement to Lesson 3.2 (Decision Trees: How AI Decides). By building one manually, students internalize how branching logic works before they ever see it in code.

---

---

## Learning Goals

By completing this activity, students will:
- Understand how yes/no questions create branching classification paths
- See how decision trees handle ambiguity and edge cases
- Connect hand-built tree logic to how AI classifiers actually work
- Practice computational thinking: decomposing a classification problem into testable conditions
