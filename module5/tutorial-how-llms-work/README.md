# Tutorial: How LLMs Work
**Module 5 | Online Tutorial | ~7 minutes**

---

## Overview

This animated tutorial demystifies large language models using a single core analogy: **an LLM is an extremely well-read autocomplete**. No math. No hype. Just the actual mechanics, explained clearly enough for a 6th grader.

Topics covered:
- **Tokens** — how words get chopped into manageable pieces
- **Context window** — what the model can and can't "see"
- **Next-token prediction** — probability, not lookup
- **Training** — where the patterns come from
- **Fluency vs. correctness** — why it sounds right but can still be wrong
- **Practical takeaways** — how to use this well

## Tone

Curious and grounded. This tutorial demystifies LLMs without either over-hyping them ("this changes everything!") or dismissing them ("it's just autocomplete"). The goal: give learners an accurate mental model they can actually use.
