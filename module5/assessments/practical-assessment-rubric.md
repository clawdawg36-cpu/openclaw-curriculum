# Module 5 Practical Assessment Rubric
## "Use OpenClaw to Explain a Real Concept"

## Overview
Students use OpenClaw to get an explanation of a real concept from **another class** (science, history, math, English, etc.), then evaluate and document the AI's response.

**Total Points: 20**

---

## Task Requirements
1. Choose a concept you're currently learning in another class
2. Craft a prompt asking OpenClaw to explain it
3. Record the AI's response
4. Verify the explanation against at least one real source
5. Write a brief evaluation (accuracy + usefulness)

---

## Rubric

### Section A: Prompt Quality (4 points)

| Score | Description |
|-------|-------------|
| **4** | Prompt names the concept clearly, specifies their grade level or background, and requests a specific format or detail level. |
| **3** | Prompt is clear and specific but missing one element (e.g., no format request or no context about their level). |
| **2** | Prompt names the concept but is vague. Output quality clearly suffers. |
| **1** | Prompt is too broad to evaluate ("explain science" / "help me with homework"). |

---

### Section B: Source Citation (4 points)

| Score | Description |
|-------|-------------|
| **4** | Student cites at least one real, credible source (textbook, .edu site, library resource, teacher) and clearly compares it to the AI's output. |
| **3** | Source cited but comparison to AI output is brief or incomplete. |
| **2** | Source mentioned but not properly cited or not clearly connected to the AI's explanation. |
| **1** | No source cited. Student only uses AI's output with no verification. |

---

### Section C: Accuracy Evaluation (8 points)

| Score | Description |
|-------|-------------|
| **8** | Student correctly identifies whether the AI's explanation is accurate, partially accurate, or inaccurate. Specific examples cited (e.g., "the AI said X but my textbook says Y"). Nuanced — recognizes that AI can be right about some parts and wrong about others. |
| **6** | Accurate evaluation with good examples, but analysis could be more specific or nuanced. |
| **4** | Student makes a general judgment ("it was mostly right") but doesn't back it up with specifics. |
| **2** | Evaluation is inaccurate or the student didn't actually check the AI's claims. |
| **0** | No accuracy evaluation included. |

---

### Section D: Reflection on AI as a Study Tool (4 points)

| Score | Description |
|-------|-------------|
| **4** | Student reflects thoughtfully on when AI *would* and *wouldn't* be a good study tool for this type of content. Makes a specific recommendation (e.g., "good for getting an overview, but always verify details"). |
| **3** | Reflection present and reasonable, but somewhat generic. |
| **2** | Brief reflection that doesn't engage with the specific content they tested. |
| **1** | No reflection, or reflection is just "it was helpful" / "it wasn't helpful." |

---

## Hallucination Watch
If the AI **hallucinated** (made up facts, names, dates, or definitions), give the student bonus recognition for catching it — even if they gave it high marks for the prompt. Finding hallucinations is a skill, not a failure.

---

## Total Score Guide

| Total | Grade Equivalent |
|-------|-----------------|
| 18–20 | A |
| 14–17 | B |
| 10–13 | C |
| 6–9 | D |
| 0–5 | Incomplete |

---

## Suggested Source Types (share with students)
- ✅ Class textbook
- ✅ Khan Academy
- ✅ School library databases
- ✅ Teacher or class notes
- ✅ Government or .edu websites
- ⚠️ Wikipedia (usable, but verify — cite the primary sources at the bottom)
- ❌ Another AI chatbot
- ❌ Random unverified websites
