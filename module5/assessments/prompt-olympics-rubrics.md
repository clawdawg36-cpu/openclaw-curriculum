# Prompt Olympics Event Rubrics
## Module 5 — Lesson 5-3

---

## Event Overview

The Prompt Olympics has 4 events:
- 🗳️ **Class-Voted Events** (Most Creative, Funniest Output)
- 📋 **Teacher-Scored Events** (Best Study Guide, Best Concept Explanation)

---

## 🗳️ Class-Voted Events

### Event 1: Most Creative Output
**Scoring:** Class vote — top 3 prompts get 3 pts / 2 pts / 1 pt

**What students are voting on:**
- Did the prompt produce something genuinely creative or unexpected?
- Was the result surprising, original, or inventive?
- Would you *want* to read the full output?

**Voting procedure:**
1. Teacher (or a student volunteer) reads each submission's prompt + first 2-3 sentences of output
2. Students vote by raising hands or using a form
3. No voting for your own submission

---

### Event 2: Funniest Output
**Scoring:** Class vote — top 3 prompts get 3 pts / 2 pts / 1 pt

**What students are voting on:**
- Did the prompt produce something genuinely funny?
- Is the humor in the *output* (not just a weird prompt that produced nonsense)?
- Bonus appreciation (not extra points) for clever setups

**Voting procedure:** Same as Event 1

**Teacher note:** Keep it appropriate. If a submission is funny but inappropriate, it's disqualified — not penalized. Just skip it.

---

## 📋 Teacher-Scored Events

### Event 3: Best Study Guide

**Task:** Students write a prompt that gets OpenClaw to produce a study guide for a real topic they're learning in school.

**Rubric (12 points total)**

| Criterion | 4 | 3 | 2 | 1 |
|-----------|---|---|---|---|
| **Prompt Clarity** | Topic, scope, and format all specified | Topic and scope clear, format vague | Topic clear but scope too broad | Vague or unstructured |
| **Output Usefulness** | Study guide is accurate, organized, and genuinely useful for studying | Mostly useful with minor issues | Partially useful; missing key info or poorly organized | Not useful as a study tool |
| **Format Control** | Output format matches request (headers, bullets, sections, etc.) | Format mostly matches | Some format elements present | No evidence of format control |

**Scoring Guide:**
- 11–12: Gold 🥇
- 9–10: Silver 🥈
- 7–8: Bronze 🥉
- Below 7: Participation

---

### Event 4: Best Concept Explanation

**Task:** Students write a prompt that gets OpenClaw to explain a complex concept *to a specific audience* (e.g., "explain photosynthesis to a 5-year-old" or "explain supply and demand to my grandma").

**Rubric (12 points total)**

| Criterion | 4 | 3 | 2 | 1 |
|-----------|---|---|---|---|
| **Audience Specification** | Audience clearly defined in the prompt; output clearly adapts to that audience | Audience named but output only slightly adapted | Audience mentioned but output doesn't really change | No audience specified |
| **Accuracy** | Explanation is factually correct | Mostly correct with 1 minor error | Some inaccuracies or oversimplification that distorts meaning | Significant factual errors |
| **Accessibility** | Explanation is genuinely easy to understand for the specified audience; uses analogies or relatable examples | Mostly accessible with some jargon | Mixed — some parts accessible, some not | Too technical or too vague |

**Scoring Guide:**
- 11–12: Gold 🥇
- 9–10: Silver 🥈
- 7–8: Bronze 🥉
- Below 7: Participation

---

## Final Standings

Tally up class-vote points + teacher-scored points for each student.

| Place | Points | Award |
|-------|--------|-------|
| 1st | Most combined | 🥇 Prompt Champion |
| 2nd | Second most | 🥈 Prompt Pro |
| 3rd | Third most | 🥉 Prompt Contender |
| All | Participated | 🎖️ Prompt Olympian |

**Teacher note:** The Prompt Olympics is meant to be energizing and fun. Err on the side of celebrating creativity over strict scoring. If something is close, round up.
