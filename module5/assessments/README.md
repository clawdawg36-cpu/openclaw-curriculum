# Module 5 Assessments

Assessment materials for **Module 5: Talking to AI**

---

## Files in This Directory

### Prompting Assessments
| File | Description |
|------|-------------|
| `prompt-portfolio-rubric.md` | 4-criterion rubric (specificity, technique use, output quality, reflection depth) for student's 3-best-prompts portfolio |
| `practical-assessment-rubric.md` | Rubric for "Use OpenClaw to explain a concept from another class" — includes source citation and accuracy evaluation |
| `prompt-olympics-rubrics.md` | Event rubrics for both class-voted (creative/funny) and teacher-scored (study guide, concept explanation) events |
| `exit-tickets.md` | Daily exit ticket templates for all 6 lessons |

### Channel Connection Lab Assessment
| File | Description |
|------|-------------|
| `channel-connection-lab-rubric.md` | Pass/fail connection check + 4-point test plan quality + 4-point reflection quality |
| `test-plan-grading-guide.md` | Teacher guide — what makes a good test case (clear input, specific expected output, honest actual output) |

### Module-Level Materials
| File | Description |
|------|-------------|
| `vocabulary-list.md` | All 10 M5 terms with definitions and examples (token, context window, hallucination, API, webhook, bot, credential, prompt engineering, few-shot, chain-of-thought) |
| `student-self-assessment.md` | Student-facing self-assessment form with skills check-in, reflection, and big questions |
| `module-badge-criteria.md` | Prompt Pro badge criteria — complete 10/15 prompt challenges + pass quiz |
