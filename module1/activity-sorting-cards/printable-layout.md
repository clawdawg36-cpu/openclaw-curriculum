# Printable Card Layout — AI or Not AI?

## Overview

This guide explains how to print and prepare the 30 sorting cards for classroom use.

---

## Recommended Card Size

**3.5" × 5" (index card size)** — the most practical size for student handling.

- Fits 4 cards per standard US Letter page (8.5" × 11") with comfortable margins
- Alternatively, use **4" × 6"** for larger text (better for younger students or projected demos)

---

## Print Layout

### Option A: 4-Up Layout (4 cards per page — recommended)

Arrange cards in a 2×2 grid on each page:

```
┌─────────────┬─────────────┐
│   Card 1    │   Card 2    │
│   (Front)   │   (Front)   │
├─────────────┼─────────────┤
│   Card 3    │   Card 4    │
│   (Front)   │   (Front)   │
└─────────────┴─────────────┘
```

- Print fronts on one side, backs on the other (duplex printing)
- For duplex: flip on the **short edge** so front and back align correctly
- Total pages needed: 8 pages of fronts + 8 pages of backs = 16 pages per set

### Option B: 2-Up Layout (2 cards per page — easier cutting)

Arrange cards in a 1×2 grid, landscape orientation:

```
┌─────────────────┬─────────────────┐
│     Card 1      │     Card 2      │
│     (Front)     │     (Front)     │
└─────────────────┴─────────────────┘
```

- More whitespace, easier to cut cleanly
- Total: 15 pages per set

---

## Printing Tips

1. **Use cardstock (65–110 lb)** if possible — regular paper flops around and answers show through.
2. **Color printing** makes the emoji stand out and improves readability; grayscale works fine too.
3. **Set page margins to 0.5"** to maximize card size.
4. **Print one test page first** to confirm front/back alignment before printing the full set.
5. **Print multiple sets** — enough for pairs or small groups (1 set per 2–4 students works well).

---

## Cutting

- Use a **paper trimmer/guillotine cutter** for straight, fast cuts — much cleaner than scissors.
- Score fold lines lightly before cutting if doing duplex cards.
- Cut along the outer border lines; a 1/16" white border around each card looks clean and professional.

---

## Lamination (Highly Recommended!)

Laminating the cards dramatically extends their lifespan for repeated classroom use.

**Steps:**
1. Print and cut all cards first
2. Run through a **thermal laminator** using 5 mil pouches
3. Let cool completely before re-cutting along the laminate edge
4. Store in a small zip-lock bag or rubber-banded by set

**Tip:** A set of laminated cards can last years with classroom use. If you have a class set of 30 students in pairs, print 15 sets and laminate them all at once — batch processing saves time.

---

## Storage & Organization

- Store each set of 30 cards in a labeled zip-lock bag or small box
- Label each bag: **"AI or Not AI? — Module 1 Sorting Cards"**
- Include a small card listing: ✅ AI cards: 1–15 | ❌ Not AI cards: 16–30

---

## Classroom Setup

**For a sorting activity:**
1. Distribute 1 set of shuffled cards per student pair
2. Provide two labeled areas on the desk or table: "AI" and "Not AI"
3. Give students 5–8 minutes to sort all 30 cards
4. Students flip cards to check answers
5. Discuss any surprising ones as a class

**Suggested discussion cards:** Autocorrect (Card 6), Basic GPS (Card 29), and Elevator Buttons (Card 24) tend to spark the most debate!
