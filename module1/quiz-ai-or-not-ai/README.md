# Online Quiz — AI or Not AI?

**Module:** 1 — What Is AI? — Meet Your Digital Brain  
**Format:** Auto-graded online quiz  
**Length:** 10 questions (~10–15 min)  
**Recommended Timing:** After Lesson 1.2 or as Module 1 culminating check

## Overview

A 10-question auto-graded quiz that checks students' ability to identify AI systems, distinguish AI from non-AI, and apply core Module 1 vocabulary. Designed for use on any LMS (Google Classroom, Canvas, Schoology) or as a standalone Google Form.

## Question Breakdown

| # | Type | Topic |
|---|------|-------|
| 1–4 | Multiple choice | Identify AI from a list of technologies |
| 5–7 | True/False | Common AI myths and facts |
| 8–9 | Scenario-based | Given a description, is it AI? |
| 10 | Short answer | Name one AI you use + explain how it works |

Each question includes the correct answer and a student-facing explanation shown after submission.

## Files in This Folder

| File | Description |
|------|-------------|
| `quiz.md` | All 10 questions with answer options, correct answers, explanations, and difficulty ratings |
| `quiz-guide.md` | Passing score recommendation, retry policy, answer key, and formative assessment usage notes |

## Recommended Settings

- **Passing score:** 70% (7/10)
- **Retake policy:** Allow 1 retake after reviewing incorrect answers
- **Time limit:** None (low-stakes formative)
- **Show answers after:** Immediately — the explanations are a teaching tool
