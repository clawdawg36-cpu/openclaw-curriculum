## Answer Key (Teacher Copy)

**Q1 — Vocabulary Match:**
- Algorithm → B
- Bias → D
- Training Data → A
- Natural Language Processing → C
- Computer Vision → E

**Q2 — Short Answer (sample):**  
AI is a broad field focused on building systems that can perform tasks that normally require human intelligence. Machine learning is a *technique within AI* where systems learn from data and improve over time without being explicitly programmed for every scenario.

**Q3 — Multiple Choice:** B  
The Turing Test specifically measures whether a machine's conversational behavior is indistinguishable from a human's.

**Q4 — Strengths & Limitations (examples):**
- Does well: recognizing faces in photos; translating text between languages; recommending movies based on watch history; detecting spam
- Struggles: understanding sarcasm; making moral judgments; reasoning about completely new situations; physical tasks requiring dexterity
