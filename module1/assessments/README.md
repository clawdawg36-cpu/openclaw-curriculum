# Assessments — Module 1

**Module:** 1 — What Is AI? — Meet Your Digital Brain  
**Assessment Types:** Formative (exit tickets, reflection journal) + Summative (formative check)

## Overview

This folder contains all assessment materials for Module 1. Assessments are designed to be low-stakes and formative — the goal is to check understanding and guide instruction, not to grade students on content they're just encountering for the first time.

## Files in This Folder

| File | Description |
|------|-------------|
| `exit-tickets.md` | 4 exit tickets (one per lesson), 3–5 questions each — quick end-of-class checks |
| `reflection-journal-rubric.md` | 4-point rubric for evaluating reflection journals, with sample student responses at each level |
| `module1-formative-check.md` | 5-question culminating check covering key Module 1 vocabulary and concepts |
| `vocabulary-list.md` | 12 key terms with student-friendly definitions and real-world examples |

## Exit Ticket Summary

| Lesson | Key Check |
|--------|-----------|
| 1.1 — What Is AI? | Define AI in your own words + name 1 AI system |
| 1.2 — AI in Your World | Name 3 AI systems from your life + which you'd miss most |
| 1.3 — First Contact | Describe your OpenClaw experience + one thing that surprised you |
| 1.4 — Turing Test | 2 things AI does well + 2 things it can't do |

## Key Vocabulary (Module 1)

AI, machine learning, algorithm, training data, neural network, natural language processing (NLP), computer vision, Turing Test, automation, bias, dataset, model

Full definitions with examples are in `vocabulary-list.md`.

## Grading Guidance

- Exit tickets: ungraded — use for next-day warm-up discussion
- Reflection journal: grade with rubric in `reflection-journal-rubric.md` — suggest grading 2–3 entries per student per module
- Module 1 formative check: suggest 70% passing threshold, allow retake
