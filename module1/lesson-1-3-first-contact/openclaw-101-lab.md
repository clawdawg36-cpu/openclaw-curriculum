## How This Works

You're going to run 10 experiments with OpenClaw. For each challenge:
1. Read the instructions
2. Try the suggested prompt (or make it your own)
3. Write down what OpenClaw said (summarize — you don't need to copy everything)
4. Answer the reflection question

There are no wrong answers here. Be curious. Be honest. Have fun.
