# AI Explorers: OpenClaw/AI Curriculum for Middle School

A project-based, hands-on curriculum introducing AI concepts to middle school students (grades 6–8) using OpenClaw as the primary learning environment. Students go from "What even *is* AI?" to building real automations with a personal AI assistant.

---

## 🗺️ Module Overview

| Module | Title | Description | Grade Level |
|--------|-------|-------------|-------------|
| **M1** | What Is AI? | Foundations: history, vocabulary, first contact with OpenClaw | 6–8 |
| **M2** | How AI Thinks | Machine learning basics, pattern recognition, training data | 6–8 |
| **M3** | AI in the Real World | Applications: healthcare, climate, art, bias & ethics | 6–8 |
| **M4** | Talking to AI | Prompt engineering, conversation design, effective queries | 6–8 |
| **M5** | AI as a Tool | Automation, productivity workflows, integrations | 7–8 |
| **M6** | Building with AI | Mini-projects: build your own AI-powered assistant workflows | 7–8 |
| **M7** | AI & Society | Ethics, privacy, jobs, responsible use, looking ahead | 6–8 |

Each module folder contains:
- `README.md` — module overview, learning objectives, prerequisites
- Lesson folders (`lesson-X-Y-slug/`) — full lesson plans, slides, handouts
- Activity folders (`activity-*/`) — hands-on activities with teacher notes
- Lab folders (`lab-*/`) — guided student labs using OpenClaw
- Tutorial folders (`tutorial-*/`) — step-by-step walkthroughs
- Quiz/assessment folders — formative and summative assessments
- `teacher-guide/` — pacing guide, differentiation tips, answer keys
- `textbook-chapter-X/` — readable chapter text for students

## 🚀 Getting Started

### For Teachers
1. Start with `module1/teacher-guide/README.md` for the full pacing guide
2. Each lesson folder has a `LESSON_PLAN.md` with objectives, materials, and timing
3. Labs require students to have access to OpenClaw (or the teacher's shared instance)

### For Students
1. Start at `module1/tutorial-welcome-to-openclaw/`
2. Follow the numbered lessons in order within each module
3. Complete labs and activities as directed by your teacher

## 🤝 Contributing

We welcome contributions from teachers, students, and AI enthusiasts!

1. **Fork** this repo
2. **Create a branch**: `git checkout -b feature/your-contribution`
3. **Make your changes** — follow the existing folder/file structure
4. **Submit a Pull Request** with a clear description of what you added or changed

### Contribution Guidelines
- Keep language accessible for grades 6–8
- Include teacher notes for every lesson
- Test all OpenClaw-based activities before submitting
- Use plain Markdown — no proprietary formats
- Images go in an `assets/` subfolder within the relevant lesson folder
