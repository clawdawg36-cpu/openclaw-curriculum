## Reddit — r/homeschool

**Title:**
We built a free, open AI curriculum for middle schoolers — 7 modules, real projects, no coding experience needed

**Body:**

Hey r/homeschool — sharing this in case it's useful for your family.

My [kid/students] kept asking about AI — how it works, whether it's going to take all the jobs, whether they should be worried. I couldn't find anything that actually taught the concepts without dumbing it down or requiring a CS background to facilitate. So we built it.

**AI Explorers** is a free, open-source AI curriculum designed for grades 6–8. It's structured as a semester-long course (about 7 modules), and the capstone project is building your own working AI — not just learning *about* AI.

Here's what's covered:
- What AI actually is (and isn't)
- How machine learning works — with hands-on activities, not just videos
- Training your own image classifier or text model
- AI ethics: bias, fairness, and who's accountable
- Natural language processing — how ChatGPT-style systems work
- Building a real AI project (chatbot, classifier, or recommendation system)

**For homeschoolers specifically:**
- Self-paced — your kid can work through it independently
- No prior coding or math required
- Each module has a clear deliverable so there's something to show in a portfolio
- Everything is open source — fork it, adapt it, skip modules you don't need

It's completely free. No account required to download the materials. Everything is on GitHub if you want to audit the content before using it.

Happy to answer any questions about how to use it with different learning styles or ages. We've seen it work with kids as young as 11 and as old as 15.
