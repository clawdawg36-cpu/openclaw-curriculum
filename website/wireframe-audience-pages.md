## Overview

Five audience-specific landing pages, each with:
- Hero section
- Key value propositions
- Trust signals
- Downloadable resource CTA
- Page-specific FAQ

Visual mode per brand guidelines:
- `/homeschool` → Dark mode, mascot Patch present, emoji OK
- `/educators` → Light surface + dark nav, no mascot, minimal emoji
- `/schools` → Light surface + dark nav, no mascot, no emoji
- `/policy` → Light surface + dark nav, no mascot, no emoji
- `/parents` → Dark mode, mascot Patch present, emoji OK
