# Module 7 Badge Criteria

**AI Builder Badge** and **AI Explorer Badge** — earn them by completing the full curriculum journey.

---

## 🏗️ AI Builder Badge

> *"I built a complete AI agent — from scratch to Expo."*

### What it represents
You didn't just learn about AI. You made something. The AI Builder badge recognizes students who completed the full build arc of Modules 1–6 and presented their finished agent at the AI Expo.

### How to earn it

| Requirement | Description |
|-------------|-------------|
| ✅ Module 1 capability unlock | Agent has a name and personality |
| ✅ Module 2 capability unlock | Agent has image or audio perception |
| ✅ Module 3 capability unlock | Agent has custom domain knowledge |
| ✅ Module 4 capability unlock | Agent has a trained or fine-tuned model integrated |
| ✅ Module 5 capability unlock | Agent is connected to a live channel |
| ✅ Module 6 capability unlock | Agent has guardrails and a usage policy |
| ✅ Presented at AI Expo | Student delivered a 5-minute live demo (or backup demo) at the Expo |

### Notes for teachers
- Students who completed a subset of modules can still present at Expo — but the AI Builder badge requires **all six capability unlocks** to be documented in the Agent Design Journal
- The Expo presentation is required; quality is graded separately on the rubric in `assessments/`
- If a student's live demo fails but they deliver a coherent backup presentation, this **still counts** for the badge — preparation and presentation are the requirement, not perfect execution

---

## 🌍 AI Explorer Badge

> *"I made it all the way. Seven modules. Full journey."*

### What it represents
The AI Explorer badge is the full-year recognition for students who completed the entire curriculum: all seven modules, a complete portfolio, and every badge along the way. It represents sustained effort, growth, and a real engagement with the full scope of AI — from building to ethics to presentation.

### How to earn it

| Requirement | Description |
|-------------|-------------|
| ✅ Module 1 badge | Introduction to AI Agents |
| ✅ Module 2 badge | Perception and Input |
| ✅ Module 3 badge | Knowledge and Reasoning |
| ✅ Module 4 badge | Learning from Data |
| ✅ Module 5 badge | Connecting to the World |
| ✅ Module 6 badge | Ethics and Responsibility |
| ✅ Module 7 badge (AI Builder) | Capstone — AI Expo |
| ✅ Complete portfolio | Agent Design Journal with entries from all 7 modules |

### What the portfolio includes
A complete portfolio contains:
- A journal entry (reflection, design decisions, or learning notes) for each module
- A record of the agent's capability unlocks
- The student's Expo reflection (what they demoed, what they'd change, what they're proud of)
- Optional: evidence of any "going further" extensions (bonus activities, peer mentoring, etc.)

The portfolio doesn't need to be polished — it should be honest. The goal is a record of real growth over the full year.

### Notes for teachers
- The AI Explorer badge is awarded at the end of the year, after the Expo
- Students who earned all 7 module badges but have an incomplete portfolio can complete missing entries in a structured debrief session — portfolio entries are about reflection, not re-doing work
- Consider a special recognition moment for AI Explorer badge earners — a certificate, a shout-out at a school event, or a message to families. The full-year commitment deserves acknowledgment.

## Badge Display

Both badges should be available in digital format (PNG) for use in:
- Student portfolios / digital binders
- School LMS / grade portals
- Printed certificates

Badge asset files: `badges/ai-builder.png` and `badges/ai-explorer.png` *(to be designed and added to assets directory)*

Badge design guidelines:
- **AI Builder:** Icon suggestion — a robot with a wrench or gear; color theme: deep blue and gold
- **AI Explorer:** Icon suggestion — a compass or constellation map; color theme: deep purple and silver
- Both badges should read clearly at small sizes (48×48px) and large sizes (print-ready 300dpi)
- Include the curriculum name ("OpenClaw AI Curriculum") and module number in the badge design
