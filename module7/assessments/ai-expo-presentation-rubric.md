## Overview

This rubric assesses the student's 5-minute AI Expo presentation + 3-minute Q&A. Each criterion is scored on a 1–4 scale. Final score is weighted as shown.

---

## Scoring Criteria

### 1. Technical Completeness — 30%

*How many M1–M6 capabilities are present and working in the agent?*

| Score | Description |
|-------|-------------|
| **4** | All 6 module capabilities are present and working: personality (M1), perception (M2), knowledge (M3), trained model (M4), live channel (M5), guardrails (M6) |
| **3** | 5 of the 6 capabilities are present and working |
| **2** | 3–4 capabilities are present and working |
| **1** | Fewer than 3 capabilities are present or working |

### 2. Clarity of Explanation — 25%

*Can a non-technical audience understand what the agent does and how it works?*

| Score | Description |
|-------|-------------|
| **4** | Crystal clear — student uses plain language, relatable analogies, and examples a non-technical adult could follow; no jargon without explanation |
| **3** | Mostly clear — occasional technical terms slip through, but overall accessible; audience can follow the main idea |
| **2** | Somewhat clear — explanation is understandable to some but would lose a non-technical audience at key moments |
| **1** | Jargon-heavy or confusing — technical language is unexplained; non-technical audience would struggle to follow |

**Score (circle):** &nbsp;&nbsp; 1 &nbsp;&nbsp; 2 &nbsp;&nbsp; 3 &nbsp;&nbsp; 4

### 3. Live Demo Execution — 20%

*Does the demo work? Does it actually show what the student claims the agent can do?*

| Score | Description |
|-------|-------------|
| **4** | Flawless (or near-flawless) — 3+ capabilities demonstrated live, demo matches all claims, student handles any hiccups with confidence |
| **3** | Solid — 2–3 capabilities demonstrated, minor issues handled gracefully, demo is credible |
| **2** | Partial — demo works for some capabilities or relies heavily on screenshots/video fallback; not all claims are validated live |
| **1** | Demo failed, was not attempted, or showed something different from what was claimed |

**Score (circle):** &nbsp;&nbsp; 1 &nbsp;&nbsp; 2 &nbsp;&nbsp; 3 &nbsp;&nbsp; 4

**Notes:**

_______________________________________________

### 4. Reflection Quality — 15%

*Does the student honestly assess both strengths AND limitations of their agent?*

| Score | Description |
|-------|-------------|
| **4** | Specific, honest, and insightful — student names real limitations with examples, explains what they'd fix and why, reflection goes beyond surface level |
| **3** | Honest and mostly specific — student acknowledges limitations and has some genuine insight, though reflection could go deeper |
| **2** | Present but shallow — student mentions limitations but stays vague; reflection leans positive without much honest critique |
| **1** | Generic or only positive — student doesn't meaningfully address limitations; reflection feels rote or rehearsed without real self-assessment |

**Score (circle):** &nbsp;&nbsp; 1 &nbsp;&nbsp; 2 &nbsp;&nbsp; 3 &nbsp;&nbsp; 4

### 5. Q&A Responses — 10%

*Are answers to audience questions thoughtful, accurate, and honest?*

| Score | Description |
|-------|-------------|
| **4** | Handles tough questions confidently and honestly — including "what can your agent get wrong?" — gives accurate answers and admits uncertainty when appropriate |
| **3** | Answers most questions well — minor uncertainty is handled appropriately; mostly accurate |
| **2** | Answers some questions but struggles with harder ones — deflects occasionally or gives answers that aren't quite accurate |
| **1** | Deflects questions, gives inaccurate answers, or is visibly unprepared for Q&A |
