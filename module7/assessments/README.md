# Module 7 Assessments

This folder contains all assessment tools for Module 7 (AI Expo Capstone) and the full-year portfolio.

---

## Contents

| File | Purpose | Used By |
|------|---------|---------|
| [`ai-expo-presentation-rubric.md`](ai-expo-presentation-rubric.md) | 5-criteria x 4-point rubric for grading the AI Expo presentation | Teacher |
| [`full-year-portfolio-rubric.md`](full-year-portfolio-rubric.md) | Rubric for the complete journal + capstone reflection portfolio | Teacher |
| [`peer-evaluation-form.md`](peer-evaluation-form.md) | Warm/cool/wonder form for classmate evaluations | Students |
| [`audience-evaluation-card.md`](audience-evaluation-card.md) | 3-question feedback card for non-student Expo guests | Guests |
| [`student-choice-awards.md`](student-choice-awards.md) | Class-voted awards for standout student work | Students |
| [`final-reflection-prompts.md`](final-reflection-prompts.md) | Lesson 7.7 reflection prompts for individual student journals | Students |

---

## When to Use Each Tool

**Before the Expo:**
- Share `final-reflection-prompts.md` with students (Lesson 7.7)

**During the Expo:**
- Teacher uses `ai-expo-presentation-rubric.md` while watching each demo
- Students use `peer-evaluation-form.md` between presentations
- Guests use `audience-evaluation-card.md`
- At the end: run the `student-choice-awards.md` voting

**After the Expo:**
- Grade portfolios using `full-year-portfolio-rubric.md`
