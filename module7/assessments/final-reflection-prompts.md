## Teacher Notes

**When to use this:** Lesson 7.7, after the Expo and before final portfolio submission.

**Format:** Students can write by hand in their journals, type into a document, or respond verbally (audio/video) if that better serves their needs.

**Sharing:** Consider a read-aloud share circle where willing students read one sentence from Prompt 4. The one-sentence agent description often produces surprising and moving moments.

**Collect alongside:** Agent Design Journals (all modules) for full portfolio assessment.

**Vocabulary note:** No new vocabulary is introduced in Module 7. This module consolidates all prior learning — M1 through M6 concepts should appear organically in these reflections.
