# Workshop 7.3 — Build Your Demo Script + Slide Deck

**Duration:** 50–60 min | **Type:** Production Workshop

Students write their complete 5-minute demo script, outline their slide deck, exchange peer feedback using the Warm/Cool/Wonder framework, and receive teacher approval. No one moves to Rehearsal Day without an approved draft.
