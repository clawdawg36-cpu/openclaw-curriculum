# Lesson 7.2 — Know Your Agent: Capability Inventory

**Duration:** 50–60 min | **Type:** Structured Workshop / Individual Work

Students systematically test every capability their agent has across M1–M6, rate each as Solid/Shaky/Skip, identify their top 3 demo moments, and write an honest limitations list. Output: Capability Inventory document — their map of what they built.
