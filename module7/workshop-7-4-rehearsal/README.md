# Workshop 7.4 — Rehearsal Day

**Duration:** 50–60 min | **Type:** Active Rehearsal Workshop

Timed run-throughs in pairs (two full rounds), structured feedback on clarity and pacing, and adversarial Q&A practice where partners try to ask the toughest possible questions. The goal: make the Expo feel easy by comparison.
