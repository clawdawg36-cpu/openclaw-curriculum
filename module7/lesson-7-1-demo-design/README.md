# Lesson 7.1 — Demo Design Workshop

**Duration:** 50–60 min | **Type:** Workshop / Direct Instruction

Students learn the Hook-Build-Demo-Reflect arc, analyze real demo examples, and storyboard their own 5-minute agent demo. The core insight: a great demo is a story, not a feature list.
