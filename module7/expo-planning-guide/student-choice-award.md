# Student Choice Award: Anonymous Voting System

**Module 7 | Teacher Resource**

*The People's Choice Award is voted on by students (and optionally guests) using anonymous paper ballots. It's often the most exciting award of the ceremony — because it comes from peers.*

---

## Why Anonymous Voting?

Anonymous voting matters for a few reasons:
- Students are more honest when they can't be judged for their choice
- It prevents popularity contests from fully dominating (though peer popularity will always be a factor)
- It teaches students something real about democratic processes and anonymous balloting

Paper ballots are strongly recommended over digital (Google Forms, Kahoot, etc.) because:
- Faster to collect and count on event day
- More tangible and ceremonially satisfying
- Doesn't require students to have devices open during the gallery
- Avoids "who voted for what" tracking concerns

## The Ballot

### Simple Ballot (1 vote per student)

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│          🏆  PEOPLE'S CHOICE AWARD                  │
│                                                     │
│   Which AI agent are you voting for?                │
│                                                     │
│   Station Number: _____                             │
│                                                     │
│   (Optional) What made it stand out?               │
│   _____________________________________________     │
│   _____________________________________________     │
│                                                     │
│   DO NOT write your name. This ballot is anonymous. │
│                                                     │
└─────────────────────────────────────────────────────┘
```

*Print on quarter-sheets (4 per page). Use colored paper to distinguish from other handouts.*

---

### Alternative: Ranked Choice Ballot (More Complex)

If you want to reduce the chance of a single popular student dominating:

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│          🏆  PEOPLE'S CHOICE AWARD                  │
│         (Ranked Choice — Top 3)                     │
│                                                     │
│   Rank your top 3 AI agents:                        │
│                                                     │
│   1st Choice: Station ____                          │
│   2nd Choice: Station ____                          │
│   3rd Choice: Station ____                          │
│                                                     │
│   DO NOT vote for yourself.                         │
│   DO NOT write your name.                           │
│                                                     │
└─────────────────────────────────────────────────────┘
```

*Scoring: 1st = 3 points, 2nd = 2 points, 3rd = 1 point. Sum all points per station.*

## Voter Eligibility

**Who votes?**
- All students in the class (even the presenter — they can vote for someone else)
- Optionally: guests (parents, visiting class, community judges)

**Can presenters vote for themselves?**
No. Add a note to the ballot and announce it before distributing.

**Can judges vote?**
Optional. If judges participate in the gallery, you can give them ballots. Their votes carry the same weight as students.

---

## Distribution and Collection

### When to Give Out Ballots
- **Best:** At the welcome table when guests/students arrive
- **Alternative:** Handed out by a student helper at the start of the gallery
- **Reserve a few:** Keep extras at welcome desk for latecomers

### When to Collect Ballots
- Ballot box (shoebox, envelope box, or a decorated container) placed at welcome table or designated collection point
- Announce ballot deadline 10 minutes before gallery ends: "You have 10 minutes to visit any final stations and drop your ballot!"
- Final call: "2 minutes — last chance to drop your ballot!"
- Close ballot box before awards deliberation begins

### Counting Method

**Simple count (single vote ballot):**
1. Empty box, separate ballots
2. Tally marks per station number on a tally sheet
3. Total per station → winner has most votes
4. In case of tie: teacher chooses between tied students, or both win

**Ranked choice count:**
1. Sort ballots by first-choice vote
2. Count totals (3 pts per 1st, 2 per 2nd, 1 per 3rd)
3. Create a simple table:

| Station | 1st (×3) | 2nd (×2) | 3rd (×1) | Total |
|---------|----------|----------|----------|-------|
| 01 | | | | |
| 02 | | | | |
| ... | | | | |

*Winner: highest total points.*

## Announcing the Winner

Save the People's Choice for last — it's the most democratic award and the most dramatically satisfying to reveal.

**Suggested script:**
> "And our final award — chosen not by the judges, but by YOU — the People's Choice Award. This agent received the most votes from students, guests, and visitors. The winner is... *[pause]* ... [Student Name] for [Agent Name]! [Student Name], take a bow!"

**If it's a tie:**
> "We have a tie for People's Choice! Both [Name] and [Name] received the same number of votes from their peers. In the spirit of the Expo, we're giving both of you the People's Choice Award — because the people were clearly divided in the best way."

## Tips and Variations

**"Best Comment on a Ballot":** After counting, read 1–2 anonymous ballot comments aloud (the "why" section) — this often produces heartwarming moments. Get student permission first if comments might be identifying.

**Gamify with stickers:** Instead of paper ballots, give each guest 3 star stickers. Guests place stickers directly on station signs throughout the gallery. Visual, immediate, fun — count the stickers at the end. (Less anonymous, but more interactive.)

**Photo op:** Photograph the ballot box before opening it. It's a nice documentation moment and gives the ceremony weight.
