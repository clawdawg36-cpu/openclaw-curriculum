# Room Setup: Science-Fair Style Table Layout

**Module 7 | Teacher Resource**

---

## Overview

The AI Expo uses a science-fair station model: each student has a dedicated table or desk area where they set up their laptop and materials. Guests rotate freely through the room, stopping at stations to see demos. This format maximizes student agency — they're the expert at their station.

---

## Layout Options

### Option A: U-Shape (Best for 10–18 students)

```
┌─────────────────────────────────────────────┐
│                                             │
│  [S1]  [S2]  [S3]  [S4]  [S5]  [S6]        │
│                                             │
│                                             │
│  [S7]                        [S12]          │
│                                             │
│  [S8]                        [S11]          │
│                                             │
│  [S9]  [S10]                                │
│                                             │
│              ← Guest Entry →                │
└─────────────────────────────────────────────┘

Guests circulate inside the U. Students face inward.
Teacher floats in the center or near the entry.
```

**Best for:** Standard classrooms, science labs, library meeting rooms

---

### Option B: Two-Row Grid (Best for 20–30 students)

```
┌─────────────────────────────────────────────┐
│                                             │
│  [S1] [S2] [S3] [S4] [S5] [S6] [S7] [S8]  │
│                                             │
│  ←————————— AISLE (5 ft) ——————————→       │
│                                             │
│  [S9][S10][S11][S12][S13][S14][S15][S16]   │
│                                             │
│  ←————————— AISLE (5 ft) ——————————→       │
│                                             │
│ [S17][S18][S19][S20][S21][S22][S23][S24]   │
│                                             │
│              ← Guest Entry →                │
└─────────────────────────────────────────────┘

Guests move down aisles. Students face the aisle.
Leave 5 ft minimum between rows for comfortable flow.
```

**Best for:** Larger classrooms, cafeterias, gymnasiums

---

### Option C: Perimeter / Around-the-Walls (Best for small rooms)

```
┌─────────────────────────────────────────────┐
│  [S1] [S2] [S3] [S4] [S5] [S6] [S7] [S8]  │
│                                             │
│ [S16]       OPEN FLOOR SPACE         [S9]  │
│                                             │
│ [S15]                                [S10] │
│                                             │
│ [S14] [S13] [S12] [S11]                    │
│              ← Guest Entry →                │
└─────────────────────────────────────────────┘

Students face inward. Guests flow around the perimeter.
Works great in small classrooms where furniture can't be moved much.
```

## Each Station Setup

### Required Items
- [ ] **Laptop/tablet** — student's primary device, demo ready before doors open
- [ ] **Power source** — outlet strip or fully charged battery
- [ ] **Station number sign** — printed, folded tent-card style (see template below)
- [ ] **Student name card** — at front edge of table, visible to guests
- [ ] **WiFi connection confirmed** — test the night before or morning of

### Recommended Items
- [ ] **One-page agent summary** — printed, guest can pick up and keep
  - Agent name and description
  - What it can do (3–5 bullet points)
  - Screenshot or QR code to try it
- [ ] **Physical prop or artifact** — something tactile (printed output, a design sketch, Agent Design Journal open to a key page)
- [ ] **Demo script** — student's own notes for what to show first (Post-it on laptop back works great)

### Nice-to-Have
- [ ] Small poster or display board (similar to a science fair board)
- [ ] Before/after comparison (what the agent could do in Module 1 vs. now)
- [ ] Sticky notes for guests to write feedback
- [ ] "Try it!" sign if guests can interact with the agent directly

## Station Number Signs (Printable Template)

Print these as 4" × 3" tent cards (fold a half-sheet of cardstock):

```
┌──────────────────┐
│                  │
│   STATION  01    │
│                  │
│  [Student Name]  │
│                  │
└──────────────────┘
```

*Tip: Use large font (36–48pt), bold, so guests can spot stations easily from across the room.*

## Pre-Event Room Setup Checklist

### Day Before (if room is available)
- [ ] Arrange tables in chosen layout
- [ ] Test all outlets — run extension cords where needed
- [ ] Mark aisles with floor tape if needed for flow
- [ ] Post room map near entry so guests can orient themselves
- [ ] Set up a welcome table near the entry:
  - Guest sign-in sheet
  - Schedule handout
  - Ballots for People's Choice vote
  - Pencils/pens

### Morning of Expo (30 min before students arrive)
- [ ] Confirm layout still intact (other classes may have used the room)
- [ ] Power strips and extension cords in place
- [ ] Welcome table ready
- [ ] Station number signs placed
- [ ] Room temperature comfortable

### When Students Arrive (30 min before Expo opens)
- [ ] Students go directly to their assigned stations
- [ ] Students set up laptops, open demo, test connection
- [ ] Teacher circulates to help with tech issues
- [ ] Students do one final run-through of their opening line

## Flow and Traffic Tips

**Announce a suggested flow direction** at welcome remarks (e.g., "Start at any station — we suggest moving clockwise"). This prevents bottlenecks.

**Popularity management:** Some stations will be mobbed; others quieter. Coach students in advance: "If you have a crowd, give a quick 2-minute overview to the whole group. If it's just you and one guest, go deeper."

**Time the open gallery:** 40–45 minutes feels right for a class of 25–30. Call out "10 minutes left" so guests can catch any stations they missed.

**Teacher float zone:** Stay mobile. Your job during the gallery is to spot tech crises, redirect lost guests, and take photos — not to hover at one station.

---

## Accessibility Notes

- Ensure aisles are wide enough for wheelchair access (minimum 36 inches, ideally 60 inches)
- Station heights: if possible, mix standing-height and seated-height table setups
- Designate a "quiet corner" for students who may be overwhelmed by crowd noise — they can invite guests to come to them
