## ⏱️ Quick Start: Expo Day Checklist

**1 week before:**
- [ ] Send guest invitations (see `guest-coordination-checklist.md`)
- [ ] Confirm room booking and AV setup
- [ ] Assign judging roles

**Day before:**
- [ ] Print all materials (schedules, scoring sheets, certificates, ballots)
- [ ] Test all tech (projectors, WiFi, demo devices)
- [ ] Set up room if possible

**Day of:**
- [ ] Doors open 30 min early for student setup
- [ ] Judges arrive 15 min before start for orientation
- [ ] Run the Expo!
- [ ] Count votes, complete judging, announce awards
- [ ] Class wrap-up and reflection

---

## 🎯 Expo Goals

1. **Celebrate** what students built across 7 modules
2. **Communicate** technical work to a non-technical audience
3. **Reflect** honestly on what worked, what didn't, and what comes next
4. **Connect** students to real AI career pathways and futures
