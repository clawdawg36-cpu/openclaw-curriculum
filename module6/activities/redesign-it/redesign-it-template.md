### Root cause analysis — WHY is the system unfair?

Check all that apply:

☐ **Biased training data** — the data it learned from reflected historical inequalities  
☐ **Proxy variables** — it uses factors (like zip code) that correlate with race/gender without using them directly  
☐ **Feedback loops** — the system's decisions create the conditions that reinforce its predictions  
☐ **Missing data** — some groups are underrepresented in the data so the system works worse for them  
☐ **Wrong objective** — the system is optimizing for the wrong thing (profit instead of wellbeing)  
☐ **No accountability** — there's no way to appeal or challenge the system's decisions  
☐ **Lack of transparency** — people don't know they're being evaluated or how
