# AI Debate Format Cards
## Module 6: Make Your Agent Ethical

*Print and cut these cards. One topic card per debate group. One format card for the moderator.*

---

```
┌─────────────────────────────────────────────────────────────────┐
│  🎤 DEBATE FORMAT CARD — For the Moderator                      │
│                                                                  │
│  STRUCTURED DEBATE FORMAT (35 minutes total)                     │
│                                                                  │
│  1. OPENING STATEMENTS (2 min each, Affirmative goes first)     │
│     Affirmative: _______ min                                     │
│     Negative:    _______ min                                     │
│                                                                  │
│  2. CROSS-EXAMINATION (3 min per side)                          │
│     Affirmative questions Negative: 3 min                        │
│     Negative questions Affirmative: 3 min                        │
│     Rules: Ask one question at a time. Questioner may          │
│     interrupt only to clarify. No speeches — questions only.    │
│                                                                  │
│  3. REBUTTAL ROUND (90 sec each, Negative goes first)          │
│     Respond directly to what the other side said.               │
│     No new arguments — only respond and strengthen yours.       │
│                                                                  │
│  4. AUDIENCE Q&A (5 minutes)                                    │
│     Audience asks questions. Either side may respond.           │
│                                                                  │
│  5. CLOSING STATEMENTS (90 sec each, Affirmative goes first)   │
│     Why your side should win. End strong.                       │
│                                                                  │
│  6. AUDIENCE VOTE                                               │
│     Before: raise hand if you agree with Affirmative.           │
│     After: raise hand again. Who MOVED people?                  │
│                                                                  │
│  Moderator rules: Keep time. Stay neutral. Enforce civility.   │
└─────────────────────────────────────────────────────────────────┘
```

## Debate Scoring Rubric

*For peer evaluators or teachers. Score each speaker 1–4 on each dimension.*

| Dimension | 1 (Beginning) | 2 (Developing) | 3 (Proficient) | 4 (Exemplary) |
|-----------|--------------|----------------|----------------|---------------|
| **Argument Quality** | Opinions only, no support | Some evidence, loosely connected | Clear claims with relevant evidence | Strong claims, specific evidence, clear warrants |
| **Rebuttal** | Ignores other side | Acknowledges but doesn't engage | Responds to main points | Directly refutes, acknowledges nuance |
| **Evidence** | No facts cited | General references | Specific examples | Cited facts/studies, used accurately |
| **Clarity** | Hard to follow | Mostly clear | Clear and organized | Compelling, well-structured |
| **Respectful Engagement** | Dismissive or rude | Mostly civil | Consistently respectful | Models excellent discourse |
