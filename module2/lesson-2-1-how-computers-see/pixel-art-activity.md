# Pixel Art Activity — How Computers See

**Lesson:** 2.1 — How Computers See  
**Time:** ~20 minutes  
**Materials:** Colored pencils or markers, this sheet

---

## 🎨 Part 1: Make Pixel Art (10 minutes)

Each square in the grid below is one **pixel**. Color in the squares to create an image.

**Choose one challenge:**
- **Easy:** Use the heart template (Grid A) — just color it in
- **Medium:** Use the star template (Grid B) — add your own colors
- **Hard:** Design your own image on the blank grid (Grid C)

**Rules:**
- Each square gets ONE solid color
- No shading or gradients — pure colors only (just like a real pixel!)
- Try to use at least 3 different colors

---

## 🔢 Part 2: Encode as Numbers (10 minutes)

Now you'll do what a computer does — turn your image into numbers!

### Color Reference Chart

Use this chart to look up RGB values:

| Color | R | G | B | How to write it |
|-------|---|---|---|-----------------|
| ⬛ Black | 0 | 0 | 0 | (0, 0, 0) |
| ⬜ White | 255 | 255 | 255 | (255, 255, 255) |
| 🟥 Red | 255 | 0 | 0 | (255, 0, 0) |
| 🟩 Green | 0 | 200 | 0 | (0, 200, 0) |
| 🟦 Blue | 0 | 0 | 255 | (0, 0, 255) |
| 🟨 Yellow | 255 | 255 | 0 | (255, 255, 0) |
