# Lab 2.4 — Hugging Face Intro + Connect Vision to Your Agent

**MODULE 2 CAPABILITY UNLOCK** 🔓

This is the key lab for Module 2. Students explore Hugging Face Spaces, run real AI models, and then wire a vision model directly into their personal OpenClaw agent as a custom skill.

## Files

| File | Description |
|------|-------------|
| `lab-guide.md` | Full two-part lab with step-by-step instructions |
| `model-exploration-worksheet.md` | Students try 5 models, record results, rate each |
| `skill-creation-guide.md` | Step-by-step skill creation with code snippets |
| `agent-journal-update.md` | Template for Agent Design Journal update |
| `teacher-guide.md` | Pre-lab setup, API key management, grading rubric |

## Time

- **Part 1 — Hugging Face Exploration:** ~30 minutes
- **Part 2 — Connect Vision Skill:** ~40 minutes
- **Total:** ~70 minutes (can split across two class periods)

## Prerequisite Labs

- Lab 2.3 — Teachable Machine (students understand custom training)
- Lesson 2.5 — OpenClaw setup (students have working agents)

## Through-Line

This lab is the **Module 2 Capability Unlock**. Students should update their Agent Journey Tracker after completing this lab to mark Vision as unlocked.
