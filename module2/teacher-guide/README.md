**Module:** 2 — Give Your Agent Vision
**Audience:** Classroom teachers, including those without CS or AI background

## Overview

The complete teacher guide for Module 2. Written for educators who may be teaching computer vision and Hugging Face integration concepts for the first time. Covers everything from account setup and pacing to facilitating the key API lab and communicating with parents.

## The KEY Deliverable

**Lab 2.4 — Hugging Face Skill Integration** is the crown jewel of Module 2. When a student's agent successfully calls a Hugging Face model and returns a photo caption — that's the unlock. Everything else in the module exists to support that moment.

See Section 1 for the full Arc Context and Section 5 for the complete Lab 2.4 facilitation guide.

## Note for First-Time AI Teachers

The most technically challenging part of this module (Lab 2.4) has full debug support in Section 5. You do not need to be an API expert to teach it — you need to know the common error patterns and their fixes. The guide gives you both. When in doubt, run the whole-class debug session described in the Facilitation Tips.
