# Module 2: How Computers Perceive the World

**Status:** 🚧 In Development

**Grade Level:** 6–8  
**Prerequisites:** Module 1 — What Is AI?

## Overview

Students explore how AI systems perceive and interpret visual and audio data. Topics include how computers "see" images through pixels, how image recognition models work, hands-on training with Google's Teachable Machine, speech recognition fundamentals, and building their own perception-powered projects with OpenClaw.

## Learning Objectives

By the end of this module, students will be able to:
- Explain how computers represent and process images as data
- Describe how machine learning models recognize patterns in images and sounds
- Train a simple image classifier using Teachable Machine
- Understand how speech recognition works
- Apply perception AI concepts using OpenClaw

## Topics

- How computers see: pixels, color, and image data
- Image recognition and pattern matching
- Training your own model with Teachable Machine
- Speech recognition and audio processing
- OpenClaw in action: seeing and hearing
