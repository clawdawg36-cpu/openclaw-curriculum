# Lesson 2.3 — Teachable Machine Lab

**Module 2: Seeing & Understanding — AI Perception**  
**Duration:** 2 periods (hands-on lab)  
**Format:** Computer lab with webcam

---

## Overview

Students train a 3-class image classifier using Google Teachable Machine, experience the full ML training loop firsthand, and then contrast their results with pre-trained Hugging Face models in a structured debrief.

**Through-line connection:** *"You're about to train a mini-version of what you'll deploy to your agent."*

## Learning Objectives

1. Train a 3-class image classifier using Google Teachable Machine
2. Interpret accuracy metrics and iterate to improve model performance
3. Articulate the difference between training your own model vs. using a pre-trained model
4. Make an informed decision about which approach fits their agent project

---

## Key Question

> **Which would YOU use for your agent — Teachable Machine (you train it) or Hugging Face (pre-trained by millions of examples)?**
