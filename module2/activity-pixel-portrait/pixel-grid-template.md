## How to Read a Pixel Grid (for AI)

When a computer reads your drawing, it sees something like this for the 8×8:

```
[ 255, 220, 177, 255, 220, 177, 255, 255 ]  ← Row 1 (R values only shown)
[ 255, 220, 177, 220, 177, 177, 220, 255 ]  ← Row 2
...and so on for all 8 rows
```

Each number 0–255 represents the intensity of Red, Green, or Blue. Three numbers per pixel = RGB color. Your 8×8 drawing = **192 numbers** total (8 × 8 × 3).

> Your phone selfie? About **36 million numbers.** Same idea, way more data.
